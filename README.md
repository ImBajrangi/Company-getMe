# getMe - UI Components & Visual Effects

A library of high-end, premium UI components and experimental visual effects for modern web design.

## 🌟 Included Effects
- **Particle Morphing**: GLSL-powered transitions between complex shapes.
- **Neural Networks**: Interactive, animated background visualizations.
- **Water Distortion**: Lightweight shaders for realistic surface ripples.
- **Typography Effects**: Rolling text and stylized horizontal scrolling.
- **Cursor Interactions**: Creative trail effects and custom cursors.

## 🛠️ Tech Stack
- **Languages**: HTML, CSS, JavaScript.
- **Graphics**: WebGL, GLSL (Shaders), Three.js.
